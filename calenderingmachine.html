ELECTROLYTE FILLING MACHINE

DESCRIPTION

This equipment sets the coated electrodes (both anode and cathode) to the desired thickness by applying pressure.

FEATURES

Un/Rewinding (roll-to-roll type)
Roll temperature is adjustable
Manual gap control with taper block
Gap detection using digital dial gauge (0.001mm)

FEATURES

Dimension : 1,500mm W x 1,400mm D x 1,600mm H
Weight : Approximately 2,500kg
Power Consumption : 220V, 3Phase, 15Kw
Compressed Air : Normal 5kgf/cm2
